What is Hupothesis ?

Hupothesis is a service designed to validate your hypothesis. The tool allows you to create live polls or exams files for you to verify an idea using large scale persona sample thanks to the Internet. We make sure only humans answer your polls, no bots, no repeated answers guaranteed.
